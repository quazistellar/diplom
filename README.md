# UNIREAX - Информационная система для дистанционного обучения

## Ссылка на веб-сайт платформы

[https://unireax-moonkid.amvera.io/](https://unireax-moonkid.amvera.io/)

---

## О проекте

UNIREAX - это платформа для дистанционного обучения, предоставляющая возможность для обучения и преподавания в цифровой среде. Система предназначена для слушателей курсов, преподавателей и методистов. Имеется также роль для администратора системы.

---

## Веб-сайт

### Основные разделы веб-сайта

**Для слушателей курсов:**
- Главная страница с персональным дашбордом
- Каталог курсов с поиском, фильтрацией и сортировкой
- Детальная страница курса с описанием, рейтингом и отзывами
- Личный кабинет с историей оплат и чеками
- Страница изучения курса (лекции, тесты, практические задания)
- Профиль пользователя с возможностью редактирования
- Настройки темы (светлая/тёмная)
- Список избранных курсов
- Страница с полученными сертификатами
- Статистика прогресса обучения
- Посты-объявления курса с возможностью комментирования

**Для преподавателей:**
- Панель управления курсами
- Создание и редактирование курсов
- Добавление лекций, тестов и практических заданий
- Просмотр списка слушателей
- Проверка и оценивание работ
- Статистика по курсам
- Управление объявлениями в курсе

**Для методистов:**
- Панель управления с общей статистикой по всем курсам
- Создание и редактирование любых курсов
- Управление заявками преподавателей на преподавание
- Одобрение или отклонение заявок с отправкой уведомлений на почту
- Удаление преподавателей с курсов
- Просмотр детальной информации по каждой заявке
- Управление лекциями, тестами и практическими заданиями в любых курсах
- Конструктор тестов с поддержкой всех типов вопросов
- Просмотр статистики по курсам (количество слушателей, прогресс, рейтинг)
- Экспорт статистики в CSV и PDF
- Просмотр популярных курсов с рейтингами
- Фильтрация заявок по курсам и статусам
- Поиск заявок по фамилии, имени или почту преподавателя

**Для методистов:**
- Панель администратора и доступ к ней
- Просмотр логов системы
- Управление таблицами базы данных
- Одобрение/отклонения заявок на регистрацию в качестве преподавателя или методиста

## Мобильное приложение

Помимо веб-сайта, платформа имеет мобильное приложение для слушателей курсов, разработанное на Flutter. Приложение синхронизировано с веб-аккаунтом и предоставляет весь основной функционал для обучения на мобильных устройствах.

## Функциональные возможности веб-сайта

### Для слушателей курсов:
- регистрация и авторизация,
- восстановление пароля через email,
- просмотр каталога курсов с поиском и фильтрацией,
- запись на платные и бесплатные курсы,
- оплата курсов через ЮKassa,
- просмотр детальной информации о курсе,
- добавление и удаление курсов в избранное,
- изучение материалов курса (лекции, тесты, практические задания),
- скачивание материалов лекций,
- прохождение тестов с автоматической проверкой,
- просмотр результатов тестирования с детальным разбором,
- сдача практических заданий с прикреплением файлов,
- просмотр оценок и обратной связи от преподавателя,
- просмотр постов-объявлений курса,
- добавление, редактирование и удаление комментариев,
- просмотр прогресса обучения по каждому курсу,
- просмотр общей статистики,
- получение сертификатов после завершения курса,
- просмотр и скачивание сертификатов,
- добавление отзывов на курсы (от 1 до 5 звёзд),
- редактирование и удаление собственных отзывов,
- редактирование профиля (ФИО, почта, имя пользователя),
- смена пароля,
- смена темы интерфейса (светлая/тёмная),
- просмотр истории оплат с чеками в формате PDF,
- деактивация аккаунта,
- выход из системы.

### Для преподавателей:
- просмотр списка созданных и ведомых курсов,
- создание новых курсов,
- редактирование информации о курсе,
- добавление, редактирование и удаление лекций,
- добавление, редактирование и удаление практических заданий,
- добавление, редактирование и удаление тестов,
- конструктор тестов с различными типами вопросов,
- просмотр списка слушателей курса,
- просмотр детального прогресса каждого слушателя,
- проверка и оценивание практических работ,
- комментирование работ слушателей,
- приём и отклонение работ на доработку,
- управление объявлениями в курсе,
- публикация постов и комментариев,
- просмотр статистики по курсам,
- экспорт статистики в CSV и PDF,
- массовое добавление слушателей через CSV,
- генерация паролей и аккаунтов для слушателей с последующей отправкой уведомлений на email.

### Для методистов:
- просмотр панели управления с общей статистикой (количество курсов, слушателей, завершивших обучение, средний процент завершения),
- создание новых курсов,
- редактирование любых курсов,
- удаление курсов,
- добавление, редактирование и удаление лекций в любых курсах,
- добавление, редактирование и удаление практических заданий,
- добавление, редактирование и удаление тестов,
- конструктор тестов с поддержкой всех типов вопросов (один ответ, несколько ответов, текстовый ответ, сопоставление),
- управление заявками преподавателей:
  - просмотр всех заявок на преподавание,
  - фильтрация заявок по курсам и статусам,
  - поиск по фамилии, имени или почте преподавателя,
  - одобрение заявок с автоматическим добавлением преподавателя на курс и отправкой email-уведомления,
  - отклонение заявок с указанием причины и отправкой email-уведомления,
  - удаление преподавателей с курсов,
- просмотр детальной информации по каждой заявке,
- просмотр списка слушателей курсов,
- просмотр прогресса слушателей,
- просмотр статистики по каждому курсу (количество записавшихся, завершивших, процент завершения, средний рейтинг),
- просмотр топ-10 популярных курсов по количеству слушателей и рейтингу,
- экспорт статистики в CSV и PDF,
- просмотр и редактирование собственного профиля,
- смена пароля,
- деактивация аккаунта.

- интернет-соединение,
- для мобильного приложения: Android-устройство, ~100 МБ свободного места.
